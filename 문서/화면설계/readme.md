# 프로젝트에 적용한 DB

* 해시태그 - mongodb 
* 댓글 좋아요 - mysql
* 댓글 - 회원테이블과 피드 테이블과 관계가 맺어져 있고 조인연산이 필요하여 mysql로 구현
* 피드 - SNS 특성상 빠른 피드조회 연산이 가능하도록 mongodb를 사용
* 피드 좋아요 - mongodb
* 개인회원 - mysql
* 기업회원 - mysql
* 팔로워 - mysql
