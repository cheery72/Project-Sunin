# 와이어프레임 작성

## 서비스를 개발하기 전 기획 고민

1. 회원가입

2. 로그인 후 -> 메인페이지

3. 회원정보수정

4. 프로필

5. 팔로워, 좋아요 기능

  - 팔로잉 / recent /  - 팔로워한 소식들

6. 개인 

    - 추천 알고리즘 (최신순, 좋아요순)

7. 회사 

    - 카테고리(전체, 이벤트, 제품홍보 등으로 나눠서 보여주는데) 이안에서 최신순, 좋아요순, 무작위로 정렬가능

8. 서치바 - 계정검색, 태그검색

9. 참여 게이지 & (추가기능)
